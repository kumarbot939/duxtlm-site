## Koyeb Deploy

1. Go to https://app.koyeb.com
2. Sign up with GitHub (kumarbot939)
3. Click "Create App"
4. Select GitHub as deployment method
5. Select `kumarbot939/duxtlm-site`
6. Click "Deploy"
7. App will be live at `duxtlm.koyeb.app`

Automatic deployments on every push to main branch.
